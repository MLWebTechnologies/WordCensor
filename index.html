Wordcensor

Plugin to filter bad words from site content

View project on GitHub

NEW! Joomla! 1.5x -3.x!! The WordCensor plugin is used to filter inappropriate words from content and from other modified components.  WordCensor allows automatic filtering of all content.  Features include:

\r\n
    \r\n
  • Replacement of word or all text.
    • \r\n
  • Configurable advanced filtering option.
